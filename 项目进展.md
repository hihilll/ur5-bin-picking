## 1. 项目是什么
UR5 机械臂 + 奥比中光 Gemini2 相机，从料框抓取**无序混摆的 3D 打印零件**，
放到指定位置（含**高精度定向装配**）。

## 2. 关键决策（已敲定）
- **零件**：用户自行 3D 打印 → **有 CAD 模型** → 走"基于模型的位姿估计"，非无模型抓取。
- **框架**：Ubuntu 22.04 + **ROS2 Humble**。组件：ur_robot_driver / OrbbecSDK_ROS2 / MoveIt2 / easy_handeye2。
- **相机布局**：Eye-to-Hand（固定俯视料框）。
- **夹爪**：沃姆 **EPGC-50-150-B-C-NM-L2**（行程50mm、负载3kg、**USB/Modbus** 独立通信）。
- **UR5 IP**：暂定 `192.168.1.102`（用户待确认真实值）。
- **路线分三阶段**：①传统 PPF+ICP（当前已实现）→ ②大模型(SAM2/FoundationPose/AnyGrasp) → ③具身(Diffusion Policy/π0)。
- **运动规划**：自由移动 RRTConnect + 抓取接近/抬起用笛卡尔直线 + 料框碰撞避障。
- **高精度装配**：在手位姿重估计 + 放置补偿（混合方案：学习/几何分工）。
- **开发工作流**：在 **Windows**(`E:\zhuomian\robotarm`) 写代码 → 拷到**另一台 Ubuntu 双系统机**构建测试。

