# 调研记录：抓取策略（Grasp Synthesis / Grasping Policy）论文与方法综述

> 用途：抓取策略方向的研究汇报底稿。2026-07 调研（含网络检索补充至 2026 年中）。
> 与 `docs/11`（发论文选题+就业）互补：本文聚焦**方法学全景**——抓取策略这个问题
> 是怎么定义的、有哪些技术路线、各路线代表作、评测基准，以及与本项目三条路线的对应。

---

## 1. 问题定义与范围

**抓取合成（grasp synthesis / grasp detection）**：给定感知输入（点云/RGB-D/图像），
输出夹持器位姿（4-DoF 平面矩形 或 SE(3) 6-DoF）+ 开口宽度 + 质量分数。

**抓取策略（grasping policy，更大范围）**：除"往哪抓"外，还包括——
抓取顺序决策（先抓谁）、辅助动作（推/拨开）、闭环伺服与失败恢复、
抓取-放置耦合（为了放得准而选择怎么抓）。工业 bin picking 的完整策略
横跨这两层，汇报时建议按"抓取合成 → 抓取执行策略 → 抓取-放置一体化"三层展开。

## 2. 技术路线分类学（汇报主线）

### 2.1 分析法（Analytic，~2015 前主流，至今仍是理论基座）

基于接触力学显式判定抓取质量：力闭合(force closure)/形闭合、
Ferrari-Canny ε-quality、抓取矩阵与 GWS（Grasp Wrench Space）。
工具：GraspIt!、OpenGRASP。
- 优点：可解释、有物理保证、不需要数据。
- 缺点：需要精确物体模型与接触点，对感知噪声敏感；杂乱堆叠场景难以应用。
- **与本项目的关系**：阶段一"CAD 预标注抓取点 + 6D 位姿配准"本质是
  分析法思路的工程化（人替代算法做了质量判定）。

### 2.2 数据驱动——平面 4-DoF 抓取（2015~2020 热点，现为轻量基线）

自顶向下视角，输出图像平面内的旋转矩形 (x, y, θ, w)。
- **GG-CNN**（Morrison, RSS 2018）：全卷积逐像素预测抓取质量/角度/宽度，
  轻到能 50Hz 闭环伺服——闭环抓取的开山代表。
- GR-ConvNet、GKNet 等后续改进。数据集：Cornell、Jacquard。
- 局限：只能俯视垂直抓，无法利用 3D 几何。
- **与本项目的关系**：我们的 `model_free_grasp`（俯视 PCA 短边抓取）
  是这一类的无学习版本；GG-CNN 类是它的直接可替换升级。

### 2.3 数据驱动——采样-评价式（sampling-evaluation）

先生成大量候选，再用学习的质量网络打分排序。
- **GPD**（ten Pas, 2017）、**PointNetGPD**（Liang, ICRA 2019）。
- **Dex-Net 2.0/4.0**（Mahler, Berkeley, RSS 2017 / Sci. Robotics 2019）：
  用**解析质量指标在仿真里自动生成数百万标注**（670 万深度图）训练 GQ-CNN——
  "仿真+解析教师造数据"范式的鼻祖，Dex-Net 4.0 扩展到吸盘+夹爪复合策略。
- 优点：候选质量可控；缺点：两阶段慢，候选覆盖率受采样启发式限制。

### 2.4 数据驱动——端到端 6-DoF 点云抓取（2020~今，当前工业界主流）

直接从场景点云回归全场景 6-DoF 抓取分布。**本项目阶段二 grasp_client 对接的就是这一族。**
- **GraspNet-1Billion**（Fang, CVPR 2020）：里程碑数据集+基线（97 真实场景、
  >10 亿标注抓取），确立 AP@friction 评测协议；graspnet-baseline 即我们预写 wrapper 的对象。
- **GSNet**（ICCV 2021）：提出 graspness（逐点可抓性）几何度量，先筛点再回归，
  精度大幅提升，成为后续方法的通用组件。
- **Contact-GraspNet**（NVIDIA, ICRA 2021）：把抓取参数化为场景表面接触点，
  简化学习空间，杂乱场景高效生成。
- **AnyGrasp**（Fang, T-RO 2023）：GraspNet 系集大成，稠密时空抓取+碰撞感知，
  自称人类水平；**只发 SDK 不开源训练代码，商用需 license**（本项目因此优先备选下面两个）。
- **HGGD**（Chen, RA-L 2023）：2D 热图定位 + 3D 局部几何细化的两级式，速度快，
  开源可自训——**本项目阶段二首选**。
- **EconomicGrasp**（iSEE Lab, ECCV 2024）：主打"经济性"——训练资源大幅缩减
  （1/4 数据即超 SOTA），开源——**本项目阶段二备选**。
- 近两年增量：REGNet V2（不同尺寸夹爪泛化）、FlexLoG/"Rethinking 6-DoF"
  （区域聚合+局部抓取头的灵活框架）、边缘设备高效化（分层热图）、
  点云序列化注意力（Pattern Recognition 2025）、ZeroGrasp/GraspFast（2025，效率与零样本）。

### 2.5 生成式抓取（2023~今，学术热点）

把抓取合成看作**条件生成**问题，学 SE(3) 上的抓取分布而非回归单点。
- VAE 起点：6-DoF GraspNet（NVIDIA, ICCV 2019）。
- **扩散模型成为主流**：
  - **GraspGen**（NVIDIA, 2025, arXiv:2507.13097）：扩散生成 + "on-generator 训练"
    的判别器（在生成器自己的样本上训练打分器），仅仿真训练即可真机泛化；
  - **GraspGen-X**（CVPR 2026）：扩到 20 亿抓取、跨夹爪 zero-shot——
    "抓取基础模型(grasp foundation model)"的代表；
  - **DexGrasp Anything**（CVPR 2025）：物理约束注入扩散训练/采样（灵巧手）；
  - EvolvingGrasp（2025）：偏好对齐微调抓取生成器。
- 综述：Diffusion Models for Robotic Manipulation: A Survey（2025, arXiv:2504.08438）。
- 汇报亮点：多峰分布建模能力（同一物体多种合法抓法）是回归式方法做不到的。

### 2.6 语义/任务导向与语言条件抓取（2024~今，最前沿）

不只"能抓住"，还要"按任务/指令抓对部位"。
- **任务导向**：affordance 驱动（GraspAff 等）、任务约束抓取。
- **语言条件**：**ThinkGrasp**（2024）：VLM 推理 + 杂乱场景策略性部位抓取；
  SOFAR（2025）：语言接地的朝向推理；OmniDexVLG（2025）。
- **VLA 一体化**：π0 / RT-2 / OpenVLA / Dex-GraspVLA（2025）等把抓取隐式融进
  端到端视觉-语言-动作模型；DreamVLA、Dita（扩散 Transformer 策略）等 2025-26 新作。
  综述：VLA Models for Robotics: Review Towards Real-World Applications（2025）。
- **与本项目的关系**：即项目阶段三（LeRobot/ACT → π0 微调）的文献土壤；
  汇报可作为"演进终点"呈现，但注意工业精放场景 VLA 精度仍不足，
  主流结论是 **VLA 管泛化、几何管精度** 的混合架构。

### 2.7 抓取执行策略层（闭环、推-抓协同、失败恢复）

- **闭环伺服**：GG-CNN 实时重规划；动态场景重建（You Only Scan Once, 2024）。
- **推-抓协同**：开山作 VPG（Zeng, IROS 2018, Push-Net+Grasp-Net 双 FCN 学 Q 值）；
  2024-25 延续：位置感知推抓协同（CAAI TIT 2024）、双臂密集杂乱推抓
  （arXiv 2412.04052）、模糊奖励加速（Actuators 2024）、
  Ground4Act（2024，VLM 引导协同推抓）。
- **失败恢复**：检测抓空/掉落后重试或换策略（我们 pick_loop 的重试 + 夹爪
  工况寄存器"夹住/掉落"反馈即此层的工程实现）。

### 2.8 抓取-放置一体化与高精度放置（工业价值最高、学术相对蓝海）

"抓得起来"到"放得精准"之间的鸿沟——**本项目阶段四与论文选题 1 所在**。
- **在手位姿重估计**：抓起后重新估计物体相对夹爪的位姿再补偿放置。
  SimPLE（MIT, Sci. Robotics 2024）：视触觉 pick-localize-regrasp-place 全链路；
  through-hole 元件在手估计+引脚检查（2022）。
- **再抓取(regrasping)**：双臂再抓取主动降低位姿不确定性（arXiv 2503.22240, 2025）、
  不稳定中间放置的 place-and-pick 再抓取。
- **面向放置的抓取选择**：Precise Object Placement with Pose Distance Estimations
  （2021）：按"该抓法放置误差多大"来选抓取；概率化抓后位移估计（2019）。
- **对称/不确定性感知**：PS6D（2024，对称感知 bin picking 位姿）、
  点云旋转/反射对称的位姿分布估计（2025）、UA-Pose（CVPR 2025，不确定性感知位姿）。
- 汇报观点：这一层"抓取分数最高 ≠ 放置最准"，把放置误差纳入抓取选择目标
  是清晰的创新缺口（对应 docs/11 选题 1）。

## 3. 工业 bin picking 专线（与本项目最贴近）

两条正交路线（本项目两条都实现了）：
1. **基于模型（6D 位姿 → 预标注抓取）**：适合已知 CAD 的工业件，能定向放置。
   传统：PPF（Drost, CVPR 2010）+ ICP（= 我们阶段一）；
   学习：BOP 挑战赛系（GDRNPP 等）→ 零样本：**SAM-6D**（CVPR 2024）、
   **FoundationPose**（CVPR 2024）（= 我们阶段二位姿路线）；
   系统级：Pickalo（2026）：低成本硬件 + SAM-6D 的完整 bin picking 管线（与我们架构同构，可作对标）。
2. **无模型（场景级抓取检测）**：GraspNet 系（= 我们 grasp_client 路线），
   不能定向放置，适合清框、未知物。

**基准**：XYZ-IBD（2025，进入 BOP Challenge 2025 工业赛道）：真实工业复杂度
（反光、密堆、毫米级标注）——汇报里提它可展示对领域前沿基准的了解；
另有 MetaGraspNet、Fraunhofer 工业 6D 数据集。

## 6. 研究缺口与可讲的创新点（衔接 docs/11 选题）

1. **面向放置精度的抓取选择**（缺口最清晰）：现有 6-DoF 方法只优化"抓得稳"，
   不建模"这个抓法放置时误差多大"。对称工业件 + 单臂 + 在手主动消歧 = docs/11 选题1。
2. **经典管线当教师的真机数据引擎**：Dex-Net 用解析教师造仿真数据，我们可用
   阶段一/二管线当教师**在真机上**自动采集（抓取结果自动标注成败），蒸馏快策略
   = docs/11 选题2；与 2025 生成式方法的 on-generator 训练思想呼应。
3. **密堆小件的推-抓协同与恢复**（选题3）：2024-25 推抓文献均在桌面散乱场景，
   料框壁约束下的推-抓少有人做。
4. 趋势判断（汇报结尾用）：抓取基础模型（GraspGen-X 20 亿抓取、跨夹爪）+
   VLA 语义层 + 几何精放层的**分层混合架构**，是工业落地的共识方向。

**P8 年代③ 采样-评价(2016-19)**：两步走——几何启发式采样候选 + CNN 打分。
GPD(多视角投影) → PointNetGPD(直接吃爪内点云)。
局限：慢、采样启发式限制上限。配图：GPD 流程图。

**P9 年代③ 关键范式页——Dex-Net**：标注从哪来？用**解析法当老师**在仿真批量算
670 万样本训 GQ-CNN；4.0 版吸盘+夹爪混合每小时 300+ 拣选(Sci. Robotics 2019)。
讲稿提示：这页埋伏笔——"教师造数据"即我们选题 2 的思想源头。

**P10 年代④ 端到端 6-DoF(2020-23，工业现役主流)**：引爆点 GraspNet-1Billion
(97 真实场景/10 亿标注/AP@μ 统一协议)。配图：数据集示例+评测协议示意。

**P11 年代④ 方法演进对比表**：baseline(先方向后细节) → GSNet(graspness 先筛点)
→ Contact-GraspNet(锚定可见表面接触点) → HGGD(2D 热图+3D 细化,快+开源)
→ AnyGrasp(强但闭源 license) → EconomicGrasp(1/4 成本超 SOTA)。
每行一句"解决了什么"。局限收尾：回归式丢多解分布→引出生成式。

**P12 年代⑤ 生成式(2023-26)**：把抓取当条件生成 P(位姿|场景)；VAE 起点 →
SE(3) 扩散(流形上去噪)。优点：多峰、可无限采样、适合下游过滤。
配图：同一物体多个采样抓取的可视化。

**P13 年代⑤ GraspGen 与抓取基础模型**：on-generator 训练(判别器专学生成器
自己的错误模式，类 RLHF)；GraspGen-X(CVPR26)=20 亿抓取跨夹爪 zero-shot——
"grasp foundation model"成立。

**P14 年代⑥ 语义/VLA(2023-26)**：两条路径——VLM 大脑+几何手(ThinkGrasp:
GPT-4o 推理抓哪个部位) vs 端到端 VLA(π0/Dex-GraspVLA)。
共识：VLA 管理解与泛化、几何管精度、控制管最后一毫米=分层混合。

**P15 平行支线① 执行策略**：VPG(2018 推抓协同开山) → 双臂推抓/VLM 引导(2024-25)；

(夹住/掉落)+pick_loop 重试即此层工程实现。

**P16 平行支线② 抓取-放置一体化（缺口伏笔页）**：2019 抓后位移分布 → 2021 按

强调：每年仅 1-2 篇、从未成热点、工业价值极高。

## 9. 分年代方法详解（讲稿版，对应 PPT P6~P16）

### 9.1 分析法时代（1985~2015）："用力学证明抓得住"
抓取=纯力学问题。**力闭合**：接触力(含摩擦锥)能否抵抗任意方向外力/力矩；
**Ferrari-Canny ε**：所有接触力可产生的力旋量集合(GWS)的最大内切球半径
="最坏方向外力下的余量"。工具 GraspIt!(2004)。
衰落于三个不成立的假设：①要精确物体模型 ②要精确接触点(相机 mm 级噪声 vs
接触分析 mm 级敏感) ③杂乱遮挡无法建模。
→ 本项目阶段一"CAD 人工标注抓取点"= 分析法的工程化(人脑替代 ε 做质量判定)。

### 9.2 平面 4-DoF 时代（2014~2019）："把抓取变成检测框"
深度学习检测爆发后，抓取降维成图像问题：俯视矩形 (x,y,θ,w)。
Lenz(2015) 滑窗级联 CNN + Cornell 基准；**GG-CNN(RSS 2018)** 全卷积逐像素
输出质量/角度/宽度三张图，6 万参数 19ms → 首次实现 50Hz **闭环抓取**
(边下降边重预测，物体被碰歪能实时修正)——"开环到闭环"意义大于精度。
局限：只能垂直抓、丢 3D 结构。→ 本项目 model_free_grasp = 此思路的无学习版。

### 9.3 采样-评价时代（2016~2019）："候选生成+学习打分"
6-DoF 端到端学不动 → 拆两步：几何启发式采样候选 + 网络打分。
GPD(2017)：法向采样数千候选→爪内点云多视角投影→CNN 判力闭合；
PointNetGPD(2019)：打分器换 PointNet 直接吃点云。
**Dex-Net 2.0(RSS 2017)** 解决"标注从哪来"：**解析法当老师**，仿真对 1500 CAD
用 ε 指标批量算 670 万样本训 GQ-CNN；4.0(Sci.Robotics 2019) 吸盘+夹爪混合
300+ PPH。局限：两阶段秒级慢；采样启发式限制上限(没采到的好抓取打分再准也没用)。
→ "教师造数据"= 论文选题 2 思想源头(老师从仿真解析换成真机管线)。

### 9.4 端到端 6-DoF 时代（2020~2023，工业现役主流）
一个网络吃全场景点云一次输出所有抓取。引爆点 **GraspNet-1Billion(CVPR 2020)**：
97 真实场景 10 亿标注 + AP@摩擦系数统一协议。方法演进逻辑(都在压 SE(3) 空间)：
baseline(先预测逐点接近方向再回归细节) → **GSNet(ICCV21)** graspness 先筛
可抓点(成为标配) → **Contact-GraspNet(ICRA21)** 抓取锚定可见表面接触点
(搜索空间压到 2D 表面) → **HGGD(RA-L23)** 2D 热图定位+3D 局部细化(快+开源,
本项目首选) → AnyGrasp(T-RO23) 集大成但闭源 license → **EconomicGrasp(ECCV24)**
1/4 训练资源超 SOTA(单卡友好,本项目备选)。
根本局限：回归/检测框架一个位置只能输出一个"最优"，抓取天然**多解**，
分布信息全丢 → 引出生成式。

### 9.5 生成式时代（2023~2026）："从预测一个到生成分布"
抓取合成 = 条件生成 P(位姿|场景)。VAE 起点(6-DoF GraspNet, ICCV19)；
扩散接管：SE(3) 流形上去噪(SO(3) 分数匹配)。
**GraspGen(2025)**：扩散生成器 + **on-generator 训练**——判别器用生成器
自己的样本(仿真验证成败)训练，专学"这个生成器爱犯的错"(类 RLHF)；纯仿真训
直接真机泛化。**GraspGen-X(CVPR26)**：20 亿抓取跨夹爪 zero-shot →
"抓取基础模型"成立。DexGrasp Anything(CVPR25)：物理约束注入扩散(灵巧手)。
优点：多解/按需采样/适合下游可达性碰撞过滤；问题：多步去噪慢。

### 9.6 语义/VLA 时代（2023~2026）："听懂了再抓对地方"
前五代只答"哪里物理可抓"，不答"该抓哪个部位"(递刀抓刃把柄留人)。
路径一(模块化)：VLM 大脑+几何手——ThinkGrasp(2024) GPT-4o 推理先移谁抓哪个
部位，底层仍是 6-DoF 网络；SOFAR(2025) 语言接地朝向。
路径二(端到端 VLA)：π0/RT-2/OpenVLA/Dex-GraspVLA——泛化强但 mm 级精度与
工业级可靠性不足。**共识=分层混合**：VLA 管理解泛化、几何网络管这一抓、
传统控制管最后一毫米 → 恰为本项目三阶段架构逻辑。

### 9.7 平行支线
**执行策略线**：VPG(IROS18,双 FCN 学推/抓像素 Q 值,开山) → 2024-25 双臂推抓、
VLM 引导推抓(Ground4Act)。解决"网络给不出候选时怎么办"。
**抓取-放置线(缺口所在)**：2019 抓后位移分布 → 2021 按放置误差选抓取 →
2024 SimPLE(视触觉 pick-localize-regrasp-place, Sci.Robotics) → 2025 双臂
再抓取消歧。每年 1-2 篇从未成热点，工业价值极高。

### 9.8 一句话总结（汇报金句）
范式更替的驱动力从来不是网络结构，而是**"标注从哪来"**：
分析法不要数据 → 平面时代人工标注 → Dex-Net 仿真+解析教师 →
GraspNet 真实扫描+协议基准 → 生成式仿真规模化 → VLA 互联网级预训练。
数据来源的每次升级，定义一个新时代。
