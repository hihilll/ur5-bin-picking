# 运行说明（在 Ubuntu 上）

> 前置：已完成 `docs/01_环境搭建.md`，`colcon build` 成功并 `source install/setup.bash`。

## 数据流总览

```
Gemini2 ──/points──> perception_node ──/detected_objects──> grasp_planner
                                                                  │
                                              /grasp_candidates    ▼
                                          ┌──────────────> grasp_executor
                                          │                 │  调 /pick_place/run
   handeye 静态TF (base<-camera) ──> TF2 ─┘                 ▼
                                                    MoveIt2 + gripper_driver ──> UR5/夹爪
```

---

## 分阶段调试

### 阶段1：手眼标定（先做一次）
用 easy_handeye2 走 Eye-to-Hand 流程，得到 `base_link -> camera_link` 外参，
填入 `bin_picking_bringup/config/handeye_result.yaml`。
验证：相机看到的点，机器人能走到。

### 阶段2：感知（单独测）
```bash
# 先起相机
ros2 launch orbbec_camera gemini2.launch.py
# 再起感知（指定 CAD）
ros2 launch bin_picking_perception perception.launch.py \
    cad_model_path:=$(ros2 pkg prefix bin_picking_description)/share/bin_picking_description/meshes/part.stl
# 看结果
ros2 topic echo /detected_objects
# RViz2 加 PointCloud2 + MarkerArray(/detected_markers) + TF
```
调参：`bin_picking_perception/config/perception_params.yaml`
（`voxel_size`、`workspace_min/max`、`cluster_*`、`min_fitness`）。

### 阶段3：抓取规划 + MoveIt（联调）
```bash
# 总启动（相机+UR+MoveIt+感知+抓取）
ros2 launch bin_picking_bringup system.launch.py \
    robot_ip:=192.168.0.11 \
    cad_model_path:=/abs/.../meshes/part.stl
# 触发一次取放
ros2 service call /pick_place/run std_srvs/srv/Trigger {}
```
夹爪无硬件时：把 `grasp_params.yaml` 里 `gripper_driver.simulate` 设 `true`。
无真臂时用 `sim.launch.py`（UR fake hardware + 夹爪 simulate）。
> MoveIt 现用"UR5+夹爪"组合模型（自建 `moveit.launch.py`，非官方 `ur_moveit`），
> `tcp_link=gripper_grasp_tcp`（指尖）。执行器改为标准 `move_group` 接口，
> 不再依赖 `moveit_py`。详见 `docs/13_MoveIt集成.md`。真机首次把
> `max_velocity_scaling/max_acceleration_scaling` 设 0.1~0.2 低速验证。

### 仅看抓取候选（不动机器人）
`system.launch.py enable_grasp:=false` 后单独跑 `grasp_planner`，
RViz2 看 `/grasp_markers`（绿色箭头=最佳抓取）。

## 待你填的关键参数（按硬件实测）
| 文件 | 参数 | 说明 |
|---|---|---|
| `bringup/config/handeye_result.yaml` | 全部 | 手眼标定结果 |
| `perception/config/perception_params.yaml` | `workspace_*`/`voxel_size` | 料框位置与零件尺度 |
| `grasp/config/grasp_params.yaml` | `grasp_annotations` | CAD 上的抓取点 |
| `grasp/config/grasp_params.yaml` | `place_pose` | 放置点位姿 |
| `grasp/config/grasp_params.yaml` | `tcp_link` | 装夹爪后的 TCP 连杆 |
| `grasp/config/grasp_params.yaml` | `port` | 夹爪串口号 |
| `gripper_driver.py` | `REG_*` 寄存器 | 沃姆协议手册填写 |
