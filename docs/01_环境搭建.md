# 阶段 0：环境搭建（在 Ubuntu 机器上执行）

目标：装好 Ubuntu 22.04 + ROS2 Humble + 四大组件，验证相机取流与 UR5 联动。

---

## 1. 系统与 ROS2

- 系统：**Ubuntu 22.04 LTS**
- ROS2：**Humble Hawksbill**

```bash
# 设置 locale
sudo apt update && sudo apt install -y locales
sudo locale-gen en_US en_US.UTF-8
sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8

# 添加 ROS2 apt 源
sudo apt install -y software-properties-common curl
sudo add-apt-repository universe
sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key \
  -o /usr/share/keyrings/ros-archive-keyring.gpg
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] \
  http://packages.ros.org/ros2/ubuntu $(. /etc/os-release && echo $UBUNTU_CODENAME) main" \
  | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null

# 安装 ROS2 Humble 桌面版 + 构建工具
sudo apt update
sudo apt install -y ros-humble-desktop ros-dev-tools python3-colcon-common-extensions

# 每个终端 source（可加入 ~/.bashrc）
echo "source /opt/ros/humble/setup.bash" >> ~/.bashrc
source ~/.bashrc
```

## 2. UR5 驱动（ur_robot_driver）

```bash
sudo apt install -y ros-humble-ur ros-humble-ur-robot-driver
```

机器人侧准备（UR5 示教器）：
1. 设置机器人**网络/IP**（如 `192.168.0.11`），与上位机同网段。
2. 安装 **External Control** URCap，配置上位机 IP。
3. 验证联动（上位机）：
```bash
ros2 launch ur_robot_driver ur_control.launch.py \
  ur_type:=ur5 robot_ip:=192.168.0.11 launch_rviz:=true \
  rtde_config_package_timeout:=500
# 在示教器上运行 External Control 程序后，RViz2 中机械臂应随实物联动
# rtde_config_package_timeout:=500 —— 网络延迟大时防止 RTDE 握手超时导致节点 SIGABRT
```

## 3. Gemini2 相机驱动（OrbbecSDK_ROS2）

```bash
# 依赖
sudo apt install -y libgflags-dev libgoogle-glog-dev \
  ros-humble-image-transport ros-humble-image-publisher

cd ~/ros2_ws/src   # 或单独的驱动工作空间
git clone https://github.com/orbbec/OrbbecSDK_ROS2.git
# 安装 udev 规则（USB 免 sudo 访问）
cd OrbbecSDK_ROS2/orbbec_camera/scripts
sudo bash install_udev_rules.sh
sudo udevadm control --reload && sudo udevadm trigger

cd ~/ros2_ws
colcon build --packages-select orbbec_camera orbbec_camera_msgs
source install/setup.bash

# 启动 Gemini2（具体 launch 名以仓库为准）
ros2 launch orbbec_camera gemini2.launch.py
# 验证点云话题
ros2 topic list | grep points
ros2 topic hz /camera/depth_registered/points
```

## 5. easy_handeye2（手眼标定，阶段1用）

easy_handeye2 **没有 Humble 的 apt 包**（`apt install ros-humble-easy-handeye2` 会报
Unable to locate package），只能源码编译：

```bash
cd ~/ros2_ws/src
git clone https://github.com/marcoesposito1988/easy_handeye2.git
sudo apt install -y ros-humble-aruco-ros    # 标定板检测（calibrate_handeye.launch 用的是 aruco_ros/single）
cd ~/ros2_ws
rosdep install -iyr --from-paths src        # 补齐 easy_handeye2 的依赖（含 python3-transforms3d 等）
colcon build --packages-up-to easy_handeye2 # 会连带编出 easy_handeye2_msgs
source install/setup.bash
# 若启动 GUI 报 ModuleNotFoundError: No module named 'transforms3d'：pip install transforms3d
```
